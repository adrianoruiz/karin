# MPC Agente Doctor

## Como Executar o Projeto

### Pré-requisitos
- Node.js instalado

### Passos para Execução

1. **Iniciar o Servidor MCP**
   ```bash
   node server.js
   ```

2. **Iniciar o Assistente**
   ```bash
   node index.js
   ```

### Resultado Esperado

Após executar os comandos acima, você verá no terminal do `index.js` a resposta do assistente, por exemplo:

```
Resposta do assistente: Consulta agendada com sucesso!
```
